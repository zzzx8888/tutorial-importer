# Tutorial Importer Plugin for Xboard

## 简介
这是一个用于将 GitHub 仓库中的 Markdown 教程自动同步并导入到 Xboard 知识库的插件。支持定时同步、立即同步以及插件在线更新。

## 功能特性
- **Git 同步**: 自动拉取远程 Git 仓库的最新内容。
- **智能导入**: 解析 `SUMMARY.md` 生成目录结构。
- **图片本地化**: 自动下载文档中的图片到本地存储。
- **定时任务**: 支持每日或每周自动同步。
- **在线更新**: 直接在插件配置中检查并更新插件版本。

## 安装说明
1. 下载插件包解压至 `plugins/TutorialImporter` 目录。
2. 在后台启用插件。

## 使用教程

### 1. 配置仓库
在插件配置中填写远程仓库地址（例如 `https://github.com/zzzx8888/ppanel-tutorial.git`）和分支名称。

### 2. 立即导入
如果需要立即测试导入功能：
1. 打开插件配置。
2. 将「立即执行导入」选项设置为「是」。
3. 点击保存。
4. 系统将在 1 分钟内后台执行导入任务。

### 3. 插件更新
本插件支持在线更新功能：
1. 打开插件配置。
2. 在「插件更新操作」中选择「检查更新」，点击保存。
3. 等待约 1 分钟后，再次打开配置，查看「更新状态」栏，如果显示有新版本。
4. 在「插件更新操作」中选择「立即更新」，点击保存。
5. 等待约 1 分钟，插件将自动下载最新版本并覆盖安装。

## 开发者指南

### 发布新版本流程
本插件配置了 GitHub Actions 自动构建发布流程。当你需要发布新版本时，请按照以下步骤操作：

1. **更新版本号**: 
   修改 `config.json` 文件中的 `version` 字段，例如 `"version": "1.1.1"`。

2. **提交代码**:
   将所有修改提交到 Git 仓库。
   ```bash
   git add .
   git commit -m "chore: release v1.1.1"
   git push
   ```

3. **打标签并推送**:
   创建一个以 `v` 开头的标签（必须与 `config.json` 中的版本号对应），并推送到远程仓库。
   ```bash
   git tag v1.1.1
   git push origin v1.1.1
   ```

4. **自动发布**:
   推送标签后，GitHub Actions 会自动触发构建流程：
   - 自动打包插件代码为 `tutorial-importer.zip`。
   - 在 GitHub Releases 页面创建一个新的 Release。
   - 上传 ZIP 包作为附件。

   用户在插件后台点击「检查更新」时，插件会自动检测到这个新 Release 并提示更新。

## 常见问题
- **导入失败**: 请检查 Git 仓库地址是否公开，以及网络是否通畅。
- **更新失败**: 请确保 PHP 进程有写入 `plugins/TutorialImporter` 目录的权限。
